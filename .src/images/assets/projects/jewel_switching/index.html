How to play:

Use the mouse or the keyboard arrow keys to choose a jewel on the board.

Using the spacebar or the left mouse click, you can select the jewel you want to switch and then choose a direction or surrounding jewel.

Clicking the selected jewel again, or using the spacebar you can deselect the jewel again and navigate the board.
